# tailwind-cremonatv

## Getting Started

1. Install deps:
    ```bash
    npm i
    ```
1. Run the development server:
    ```bash
    npm run dev
    ```
1. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
